# Vue 3 Skill

## Core Directives
- **Composition API**: Always use `<script setup lang="ts">`.
- **State Management**: Use Composables in `src/composables/` for shared logic.
- **TypeScript**: Use `defineProps<{ ... }>()` and `defineEmits<{ ... }>()`.
